# Cancellations and Modifications Policy

## Cancellation Windows
- Orders can be canceled without penalty before status changes to PACKED.
- Once status is PACKED or IN_TRANSIT, cancellation is not guaranteed and may be treated as return after delivery.

## Address and Item Changes
- Address changes are allowed only before carrier label generation.
- Item quantity changes are allowed only before payment capture and allocation.
- If requested changes cannot be applied, support should offer cancel-and-reorder.

## Charge and Voucher Rules
- Partial cancellation recalculates tax and shipping based on remaining items.
- Promotional vouchers are reissued only if cancellation is due to system or fulfillment error.
- For customer-requested cancellation after dispatch, refund excludes outbound shipping unless carrier return-to-sender occurs.

## Exception Handling
- Duplicate orders created within 10 minutes with same customer and same cart value should be reviewed for accidental purchase.
- Fraud-review holds require identity verification before cancellation or modification approval.
- VIP or subscription customers can receive one-time fee waiver for late modification requests.
